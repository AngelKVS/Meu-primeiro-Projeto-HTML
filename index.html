Oque significa HTML?

Voltar para o inicio

Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

Oque isso quer dizer? Basicamente o HTML é usado para montar o esboço do nosso site. Apartir dele fazemos alterações com o CSS e Javascript. Com o HTML é possivél criar um site do zero. Utilizando TAGS, imagens, textos, parágrafos, etc.

Oque são TAGS e Atributos

Voltar para o inicio

As TAGS são comandos que vão orientar você na criação do seu website. Você pode utilizar elas para criar textos, adicionar imagens, definir tamanho e fonte de texto, entre outras opções.

Aqui temos as Tags mais utilizadas:

  • < head >: é o inicio do HTML;
  • < title >: indica o titulo;
  • < body >: o corpo do site;
  • < H1 >, < H2 > < H3 >: são tags para titulo e subtitulo;
  • < strong >: deixa o texto em negrito;
  • < i >: deixa o texto em itálico;
  • < p >: vai inserir um parágrafo;
  • < a href= >: é utilizado para adicionar link

Atributos são propriedades que uma TAG pode ter. Existem Atributos globais, que toda TAG tem. E existem Atributos especificos de cada TAG. O Atributo descreve as características do elemento, com ele é possivél personalizar tags.

Alguns atributos que podem ser utilizados:

  • Na TAG < strong >, temos o atributo Stylle="". Ele pode ser utilizado para alterar a cor do seu texto.
  • Na TAG < input > temos o atributo Type="". Nele você pode colocar uma caixa de texto, número e até cor no seu site.
  • Na TAG < img > temos o atributo src="". Esse Atributo é para colocar o link de uma imagem. Essa imagem vai aparecer no seu site.
  • No < img > temos também o atributo Width="". Ele é utilizado para definir a largura da imagem.

Esses são alguns dos comandos. Para estudo você pode acessar o site W3schools.

Como adicionar LINKS

Voltar para o inicio

Para adiconar links você vai utilizar a TAG < a > . Dentro dessa TAG temos alguns atributos, são eles:

  • href="" você copia o link e vai inserir ele entre os "".
  • Target="" você pode utilizar o < a target="_blank" que é para abrir o site em outra aba. Ou utilizar o < a target="_self" que abre o link na mesma aba.
